## 🔧 常用命令速查

### 构建项目

```bash
# 使用脚本(最简单)
./build.sh all

# 手动构建
mkdir build && cd build
cmake ..
cmake --build .
```

### 运行测试

```bash
cd build
ctest              # 运行所有测试
ctest -V           # 详细输出
ctest -R Calculator # 运行特定测试
```

### 清理项目

```bash
./build.sh clean   # 使用脚本
rm -rf build       # 手动删除
```

### 查看帮助

```bash
cmake --help       # CMake 帮助
./build.sh help    # 构建脚本帮助
```
